# Security Policy

Laravel Periscope is intended for trusted debugging environments and inherits access control from Laravel Telescope.

## Reporting Security Issues

Please report security concerns privately through GitHub Security Advisories on the project repository if available.

If GitHub Security Advisories are not enabled, contact the maintainer privately before opening a public issue.

Please do not disclose security issues publicly until they have been reviewed and, where necessary, patched.

## Supported Versions

Only the latest tagged release is supported during early development.
